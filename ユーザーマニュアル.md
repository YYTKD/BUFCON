        <h1 id="基本的な使い方">基本的な使い方</h1>
        <h2 id="step-1バフを登録する">Step 1：バフを登録する</h2>
        <p>バフとはキャラクターに適用される一時的なボーナス・ペナルティのことです。</p>
        <h3 id="バフの追加方法">バフの追加方法</h3>
        <ol>
          <li><strong>「バフ/デバフ」セクション</strong>の「+」ボタンをクリック</li>
          <li>
            <p>ダイアログの「個別追加」で以下を入力：</p>
            <ul>
              <li><strong>バフ名</strong>：効果の名称（例：ファイアウェポン、毒状態）</li>
              <li><strong>効果先</strong>：適用対象の判定・攻撃を選択</li>
              <li><strong>説明</strong>：補足説明（例：炎属性ダメージ+1d6）</li>
              <li><strong>効果</strong>：反映させるコマンド（例：+2、+1d6、-1）</li>
              <li><strong>ターン</strong>：継続ターン数（整数値のみ）</li>
              <li><strong>ラベルの色</strong>：表示色（サンプルまたはカラーピッカーで選択）</li>
              <li><strong>メモ</strong>：詳細な説明や備考（入力は任意）</li>
            </ul>
          </li>
          <li>
            <p>「追加」ボタンをクリック</p>
          </li>
        </ol>
        <p>
        <div class="admonition note">
          <div class="admonition-title">💡 ヒント</div>
          <div class="admonition-body">
            <p>効果先を選択する前に、反映先となる判定や攻撃を登録しておくと選択しやすいです。</p>
          </div>
        </div>
        </p>
        <h3 id="バフの一括追加">バフの一括追加</h3>
        <p>複数のバフを一度に登録できます。</p>
        <p><strong>記法：</strong> <code>バフ名|効果先1,効果先2|説明|効果|ターン|カラーコード</code></p>
        <pre><code>例：
ファイアウェポン|すべての攻撃|炎属性ダメージ|+1d6|3|#DF383E
ハイテンション|命中,武器A|命中・ダメージUP|+1|5|#F7821B
毒状態|すべての判定|-2||-|#62BA46
</code></pre>
        <ul>
          <li><strong>カテゴリ指定</strong>：<code>&lt;カテゴリ名&gt;</code> で開始、<code>&lt;/カテゴリ名&gt;</code> で終了</li>
          <li><strong>説明・効果・ターン・色は省略可能</strong></li>
          <li>説明のみ存在する場合、「簡易説明を表示」をONにすると一覧に表示されます</li>
        </ul>
        <h3 id="効果先の種類">効果先の種類</h3>
        <ul>
          <li><strong>なし</strong>：どの判定・攻撃にも適用しない</li>
          <li><strong>すべての判定</strong>：登録したすべての判定に適用</li>
          <li><strong>すべての攻撃</strong>：登録したすべての攻撃に適用</li>
          <li><strong>個別の判定/攻撃</strong>：特定の項目のみに適用</li>
          <li><strong>カテゴリ</strong>：カテゴリ内のすべての項目に適用（例：<code>&gt;&gt;魔法攻撃</code>）</li>
        </ul>
        <hr />
        <h2 id="step-2判定を登録する">Step 2：判定を登録する</h2>
        <h3 id="判定の追加方法">判定の追加方法</h3>
        <ol>
          <li><strong>「判定」タブ</strong>を開く</li>
          <li>「+」ボタンをクリック</li>
          <li>
            <p>以下を入力：</p>
            <ul>
              <li><strong>判定名</strong>：判定の種類（例：命中判定、知覚判定）</li>
              <li><strong>判定ロール</strong>：基本となるダイスロール（例：1d20、2d6+1）</li>
              <li><strong>カテゴリ</strong>：該当する場合は選択（例：戦闘判定、スキル判定）</li>
            </ul>
          </li>
          <li>
            <p>「追加」ボタンをクリック</p>
          </li>
        </ol>
        <h3 id="判定の一括追加">判定の一括追加</h3>
        <p><strong>記法：</strong> <code>判定名|判定ロール</code></p>
        <pre><code>例：
命中判定|2d6+1
回避判定|1d20
知覚判定|2d6
</code></pre>
        <h2 id="変数代入を使う場合-判定ロール内に-変数名-を記述しておくとバフから値を代入できます-例-2d61ボーナス">
          <div class="admonition tip">
            <div class="admonition-title">📌 変数代入を使う場合</div>
            <div class="admonition-body">
              <p>判定ロール内に <code>//変数名//</code> を記述しておくと、バフから値を代入できます。<br />
                例：<code>2d6+1+//ボーナス//</code></p>
            </div>
          </div>
        </h2>
        <h2 id="step-3攻撃を登録する">Step 3：攻撃を登録する</h2>
        <h3 id="攻撃の追加方法">攻撃の追加方法</h3>
        <ol>
          <li><strong>「攻撃」タブ</strong>を開く</li>
          <li>「+」ボタンをクリック</li>
          <li>
            <p>以下を入力：</p>
            <ul>
              <li><strong>攻撃名</strong>：攻撃の種類（例：クレイモア、ファイアボール）</li>
              <li><strong>攻撃ロール</strong>：基本となるダイスロール（例：2d6、K20@10）</li>
              <li><strong>カテゴリ</strong>：該当する場合は選択（例：近接攻撃、魔法攻撃）</li>
            </ul>
          </li>
          <li>
            <p>「追加」ボタンをクリック</p>
          </li>
        </ol>
        <h3 id="攻撃の一括追加">攻撃の一括追加</h3>
        <p><strong>記法：</strong> <code>攻撃名|攻撃ロール</code></p>
        <pre><code>例：
クレイモア|2d6
ファイアボール|K20@10
ショートボウ|1d6+2
</code></pre>
        <h2 id="変数代入を使う場合-判定と同様に-変数名-を利用できます-例-2d6ダメージボーナス">
          <div class="admonition tip">
            <div class="admonition-title">📌 変数代入を使う場合</div>
            <div class="admonition-body">
              <p>判定と同様に <code>//変数名//</code> を利用できます。<br />
                例：<code>2d6+//ダメージボーナス//</code></p>
            </div>
          </div>
        </h2>
        <h2 id="step-4コマンドを生成する">Step 4：コマンドを生成する</h2>
        <h3 id="判定のコマンド生成">判定のコマンド生成</h3>
        <ol>
          <li><strong>「判定」タブ</strong>で判定一覧から目的の項目をクリック（選択状態になる）</li>
          <li>下部の「コピペ用コマンド」欄に生成されたコマンドが表示される</li>
          <li>「コピー」ボタンでクリップボードにコピー</li>
          <li>セッションツールのチャットに貼り付けて実行</li>
        </ol>
        <p><strong>生成されるコマンド例：</strong></p>
        <pre><code>2d6+1+2 命中判定
</code></pre>
        <h4 id="目標値の設定">目標値の設定</h4>
        <p>判定タブの「目標値」セクションで設定可能：</p>
        <ul>
          <li><strong>なし</strong>：目標値なしでコマンド実行</li>
          <li><strong>&gt;=目標値</strong>：成功判定（例：&gt;=10）</li>
          <li><strong>=&lt;目標値</strong>：失敗判定（例：=&lt;5）</li>
        </ul>
        <h3 id="攻撃のコマンド生成">攻撃のコマンド生成</h3>
        <ol>
          <li><strong>「攻撃」タブ</strong>で攻撃一覧から目的の項目をクリック</li>
          <li>下部の「コピペ用コマンド」欄に生成されたコマンドが表示される</li>
          <li>「コピー」ボタンでクリップボードにコピー</li>
          <li>セッションツールのチャットに貼り付けて実行</li>
        </ol>
        <p><strong>生成されるコマンド例：</strong></p>
        <pre><code>2d6+1d6 クレイモア
</code></pre>

        <h1 id="バフの管理機能">バフの管理機能</h1>
        <h2 id="バフの有効無効を切り替える">バフの有効/無効を切り替える</h2>
        <p>バフ一覧の右側にあるトグルボタン（●○）で有効・無効を切り替えできます。</p>
        <ul>
          <li><strong>有効（青）</strong>：ジョイント時にコマンドに反映される</li>
          <li><strong>無効（半透明）</strong>：コマンド生成時に無視される</li>
        </ul>
        <hr />
        <h2 id="ターン経過">ターン経過</h2>
        <p>アクティブなバフのターン数を進める機能です。</p>
        <ul>
          <li><strong>「ターン経過」ボタン</strong>：すべてのアクティブなバフのターン数が1減少
            <ul>
              <li>ターンが0になると自動的に無効になります</li>
            </ul>
          </li>
          <li><strong>「ターンをリセット」ボタン</strong>：すべてのバフのターンを登録時の値に戻す</li>
        </ul>
        <h2 id="注意-無効になっているバフのターン数は進みませんターン経過ボタンは有効なバフのみに影響します">
          <div class="admonition warning">
            <div class="admonition-title">⚠️ 注意</div>
            <div class="admonition-body">
              <p>無効になっているバフのターン数は進みません。ターン経過ボタンは有効なバフのみに影響します。</p>
            </div>
          </div>
        </h2>
        <h2 id="バフの順序を変更する">バフの順序を変更する</h2>
        <p>バフは上から順にコマンドに反映されます。<strong>ドラッグ操作</strong>で順序を変更できます。</p>
        <pre><code>例：
1. +2（先に適用）
2. +1d6
3. -1（最後に適用）
→ ダイスロール+2+1d6-1
</code></pre>
        <h2 id="コマンドの位置が重要な場合-システムによっては演算子の位置が重要な場合がありますその場合は該当のバフを適切な位置にドラッグしてください">
          <div class="admonition tip">
            <div class="admonition-title">📌 コマンドの位置が重要な場合</div>
            <div class="admonition-body">
              <p>システムによっては演算子の位置が重要な場合があります。その場合は、該当のバフを適切な位置にドラッグしてください。</p>
            </div>
          </div>
        </h2>

            <div class="admonition-title">📝 機能説明</div>

        <h1 id="発展的な使い方">発展的な使い方</h1>
        <h2 id="変数代入機能">変数代入機能</h2>
        <p>判定や攻撃のロール内の任意の位置にバフの効果を代入できます。</p>
        <h3 id="基本的な使い方-2">基本的な使い方</h3>
        <p><strong>ステップ1：判定（または攻撃）のロール内に変数を配置</strong></p>
        <pre><code>例：判定ロール = 2d6+1+//ボーナス//
</code></pre>
        <p><strong>ステップ2：バフの効果に代入記法を使用</strong></p>
        <pre><code>効果：//ボーナス=+2
</code></pre>
        <p>すると、判定コマンドは以下のようになります：</p>
        <pre><code>2d6+1+2
</code></pre>
        <h3 id="複数の変数を使う">複数の変数を使う</h3>
        <p>1つのバフから複数の場所に代入する場合は、「,」で区切ります：</p>
        <pre><code>バフ効果：//ダイス増加=+1,//属性ダメージ=+1d6

判定ロール例：2d6//ダイス増加//+5//属性ダメージ//
↓
生成されるコマンド：2d6+1+5+1d6
</code></pre>
        <h3 id="複数のバフを同じ変数に代入">複数のバフを同じ変数に代入</h3>
        <p>同じ変数を指定したバフが複数ある場合、バフの並び順に全て代入されます：</p>
        <pre><code>バフ1：//攻撃力=+1
バフ2：//攻撃力=+1d6
バフ3：//攻撃力=-2

判定ロール：1d20+//攻撃力//
↓
生成されるコマンド：1d20+1+1d6-2
</code></pre>
        <h2 id="実用例-複雑なダメージ計算に対応-攻撃ロールk2010力ボーナスバフ1筋力アップ力ボーナス2バフ2武器強化力ボーナス1d6-こうすることで複雑な計算式を一つのコマンドで実現できます">
          <div class="admonition tip">
            <div class="admonition-title">💡 実用例</div>
            <div class="admonition-body">
              <p><strong>複雑なダメージ計算に対応</strong></p>
              <pre><code>攻撃ロール：K20@10+//力ボーナス//

バフ1（筋力アップ）：//力ボーナス==+2
バフ2（武器強化）：//力ボーナス==+1d6
</code></pre>
              <p>こうすることで、複雑な計算式を一つのコマンドで実現できます。</p>
            </div>
          </div>
        </h2>
        <h2 id="ユーザー辞書とオートコンプリート">ユーザー辞書とオートコンプリート</h2>
        <p>よく使う文字列をあらかじめ登録しておくと、入力時に自動でサジェストされます。</p>
        <h3 id="ユーザー辞書に登録する">ユーザー辞書に登録する</h3>
        <ol>
          <li><strong>ヘッダー</strong>の「⚙設定」をクリック</li>
          <li>「ユーザー辞書」を選択</li>
          <li>
            <p>「新規登録・編集」セクションで以下を入力：</p>
            <ul>
              <li><strong>文字列</strong>：登録する文字列（例：+1d6,//力ボーナス=+2）</li>
              <li><strong>カテゴリー</strong>：分類用の名前（例：代入記法、よく使う効果）</li>
            </ul>
          </li>
          <li>
            <p>「登録」ボタンをクリック</p>
          </li>
        </ol>
        <p>
        <div class="admonition note">
          <div class="admonition-title">📝 現在の機能</div>
          <div class="admonition-body">
            <p>ユーザー辞書のカテゴリーは、登録した文字列を分類・整理する機能のみとなっています。将来的には、よく使う項目を優先的にサジェストするなど、サジェスト順位の設定機能の実装を検討しています。</p>
          </div>
        </div>
        </p>
        <h3 id="オートコンプリートを使う">オートコンプリートを使う</h3>
        <p>バフのコマンド入力欄などで、以下の箇所で自動サジェストが表示されます：</p>
        <ul>
          <li><strong>バフの効果欄</strong></li>
          <li><strong>判定ロール入力欄</strong></li>
          <li><strong>攻撃ロール入力欄</strong></li>
          <li><strong>一括追加入力欄</strong></li>
        </ul>
        <p>入力中に該当する文字列があれば、ドロップダウンリストが表示されます。</p>
        <pre><code>入力：「+1」と打つ
↓
サジェスト例：
- +1d6
- +1d4
- //力ボーナス=+2
</code></pre>
        <p><strong>操作方法：</strong></p>
        <ul>
          <li><strong>↑↓キー</strong>：候補を選択</li>
          <li><strong>Enterキー</strong>：選択した候補を確定</li>
          <li><strong>Escキー</strong>：ドロップダウンを閉じる</li>
        </ul>

        <h2 id="カテゴリ機能">カテゴリ機能</h2>
        <p>バフ・判定・攻撃をグループ化して管理できます。</p>
        <h3 id="カテゴリの追加方法">カテゴリの追加方法</h3>
        <p>各セクション（バフ、判定、攻撃）の「新規カテゴリ追加」で以下を入力：</p>
        <pre><code>例：戦闘判定,スキル判定,探索判定
</code></pre>
        <p>複数追加する場合はカンマで区切ります。</p>
        <h3 id="カテゴリの使い方">カテゴリの使い方</h3>
        <p><strong>一括追加時のカテゴリ指定：</strong></p>
        <pre><code>&lt;魔法攻撃&gt;
ファイアボール|K20@10
アイススパイク|K20@10
&lt;/魔法攻撃&gt;

&lt;近接攻撃&gt;
クレイモア|2d6
</code></pre>
        <p>このように記述すると、自動的にカテゴリ分けされた状態で登録されます。</p>
        <h3 id="カテゴリを効果先として指定">カテゴリを効果先として指定</h3>
        <p>バフの効果先で <code>&gt;&gt;カテゴリ名</code> を選択すると、<strong>カテゴリ内のすべての項目</strong>にバフが適用されます。</p>
        <pre><code>例：
バフ「魔法強化」の効果先 → &gt;&gt;魔法攻撃
↓
カテゴリ「魔法攻撃」内のすべての攻撃に「魔法強化」が適用される
</code></pre>
        <h3 id="カテゴリへのジャンプ">カテゴリへのジャンプ</h3>
        <p>一覧の上部にあるインデックスボタン（📑）で、任意のカテゴリまでスクロールをジャンプできます。</p>
        <hr />
        <h2 id="メモ機能">メモ機能</h2>
        <p>バフに詳細な説明や備考を記入できます。</p>
        <h3 id="メモの記入方法">メモの記入方法</h3>
        <p>バフ追加ダイアログの「メモ欄」に自由に入力します。</p>
        <pre><code>例：
「このバフは〇〇によって付与される。
5ターン経過すると自動で解除される。
他の強化効果と重複しない。」
</code></pre>
        <h3 id="簡易説明を表示">簡易説明を表示</h3>
        <p>「☑簡易説明を表示」をONにすると、メモの<strong>1行目</strong>が一覧に表示されます。</p>
        <pre><code>メモ：
ダメージ+1d6、クリティカル時は+2
〇〇スキルによってのみ付与可能

↓ 一覧には以下のように表示
バフ名：ダメージ+1d6、クリティカル時は+2
</code></pre>
        <hr />
        <h2 id="パッケージの詳細表示">パッケージの詳細表示</h2>
        <p>バフ一覧のバフをクリックすると、詳細情報が展開表示されます。</p>
        <p><strong>表示される情報：</strong></p>
        <ul>
          <li>最大ターン数</li>
          <li>効果先（複数の場合は全て表示）</li>
          <li>コマンド内容</li>
          <li>メモ</li>
        </ul>

            </div>
          </div>
        </h2>
        <h2 id="データのリセット">データのリセット</h2>
        <p>すべての設定を初期化します。<strong>この操作は取り消せません。</strong></p>
        <ol>
          <li><strong>ヘッダー</strong>の「⚙設定」→「データ管理」を選択</li>

        <h1 id="トラブルシューティング">トラブルシューティング</h1>
        <h3 id="コマンドが反映されない">コマンドが反映されない</h3>
        <ul>
          <li>✅ 判定/攻撃がリストで選択されているか確認</li>
          <li>✅ バフが無効（半透明）になっていないか確認</li>
          <li>✅ 効果先の判定/攻撃が正しく指定されているか確認</li>
        </ul>
        <h3 id="変数が代入されない">変数が代入されない</h3>
        <ul>
          <li>✅ バフの記法が <code>//変数名=値</code> 形式になっているか確認</li>
          <li>✅ 判定/攻撃ロール内に <code>//変数名//</code> が存在するか確認</li>
          <li>✅ 変数名の綴りが一致しているか確認（大文字小文字も含む）</li>
        </ul>
        <h3 id="オートコンプリートが表示されない">オートコンプリートが表示されない</h3>
        <ul>
          <li>✅ ユーザー辞書に登録した文字列があるか確認</li>
          <li>✅ 入力中の文字列が登録内容と部分一致しているか確認</li>
        </ul>
        <h3 id="バフが消えない">バフが消えない</h3>
        <ul>
          <li>✅ ターン数が0になるまで「ターン経過」を実行してください</li>
          <li>✅ ターン数が設定されていないバフは手動で削除してください</li>
        </ul>
        <h3 id="データが保存されない">データが保存されない</h3>
        <ul>
          <li>✅ ブラウザのストレージ容量が満杯の可能性があります</li>
          <li>✅ 定期的にエクスポートでバックアップを取ることを推奨します</li>
        </ul>

        <h1 id="よくある質問">よくある質問</h1>
        <p><strong>Q: 複数の判定に異なるバフを適用したい</strong></p>
        <p>A: バフの効果先で複数の判定を選択してください。または、「すべての判定」を選択して、適用しない判定は別途バフを削除してください。</p>
        <p><strong>Q: バフの効果を計算式の中間に挿入したい</strong></p>
        <p>A: 変数代入機能を使用してください。判定/攻撃ロール内に <code>//変数名//</code> を配置し、バフの効果に <code>//変数名=値</code> と記述します。</p>
        <p><strong>Q: カテゴリ内の特定の項目だけにバフを適用したい</strong></p>
        <p>A: カテゴリではなく、個別の項目を効果先として指定してください。</p>
        <p><strong>Q: ユーザー辞書のデータをバックアップしたい</strong></p>
        <p>A: ヘッダーの「⚙設定」→「ユーザー辞書」から「ユーザー辞書をクリップボードにコピー」でエクスポートしてください。</p>
        <p><strong>Q: オフラインで使用できますか？</strong></p>
        <p>A: はい。ローカルアプリのため、ソースファイルをダウンロードした場合はオフラインでも使用できます。</p>
        <p><strong>Q: 複数キャラクターの設定を切り替えたい</strong></p>
        <p>A: 現在の設定をエクスポート→別のキャラクター用の設定をインポート、という方法で切り替えてください。</p>
